  - String escapes
  - Array/List type
  - For-in loop with iterator protocol
  - Dict/Hash/Map type

  - Runtime type checking (typeof/instanceof)

  - Anonymous functions
  - Tail-call optimisation
  - Lazy function parameters
  - Pattern matching

  - Metaclasses
  - Access modifiers (private, protected etc.)
  - Static methods & fields
